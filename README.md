# FUTURE_CS_01-Rapport d'évaluation des vulnérabilités 

Stagiaire chez future Interns en cybersécurité 

Tache 1 : Rapport d'évaluation des vulnérabilités pour un site web en ligne

Apercu du projet : 

Vous trouverez dans ce dépôt l'ensemble des livrables de ma première mission en cybersécurité pour Future Interns. Dans ce cadre, j'ai mené un audit de vulnérabilité passif (en lecture seule) sur un site d'entraînement public. L'objectif de l'exercice était de repérer les failles de sécurité classiques et d'expliquer ces risques avec un vocabulaire adapté aux besoins des entreprises. J'y propose également des solutions de remédiation concrètes, exactement comme le ferait un consultant en sécurité accompagnant une TPE/PME ou une startup

Site web ciblé : 

- URL :  http://zero.webappsecurity.com

-Portée :  uniquement les pages publiques, le balayage passif, l’analyse des en-têtes et la revue de configuration (Pas d’exploitation, pas d’attaques actives, pas de tentatives de connexion)

Outils utilisés : 
-Nmap – Numérisation de portage léger et de version service avec saisie de bannières (-sV --script banner)

-WhatWeb – Détection de piles technologiques et de composants

-curl – analyse des en-têtes HTTP
